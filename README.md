## 🎨 About Me

Hey there! I'm **Amber Younus** — a frontend developer who is obsessed with design and precision.  
I don't just *build* interfaces, I **craft** them — every pixel, every hover state, every transition is intentional.

- 🔭 Currently building projects with **React & Tailwind CSS**
- 🌱 Learning **Backend Development** — Node.js, Express & MongoDB
- 🎯 Goal: Become a full **MERN Stack Developer**
- 💜 I turn **Figma designs into clean, responsive React apps**
- 📍 Based in **Pakistan**
